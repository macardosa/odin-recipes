# odin-recipes

This project uses plain HTML to create a website showcasing a curated collection of delicious and nutritious recipes. You can explore main dishes, desserts, drinks, and more.

In this project, I practiced using HTML elements such as unordered and ordered lists to present collections of items and enumerate steps, respectively. I also included freely available images of the recipes, which are licensed for use, sharing, and modification, and inserted them into the HTML markup using the `<img>` tag.

I employed anchor elements (`<a>`) to connect the homepage with each of the recipes, ensuring a meaningful user experience and smooth navigation within the website. Unordered lists proved particularly versatile, helping to display links on separate lines, since `<a>` elements are inline by default.

As a safety measure, I used minimal CSS to guarantee that the images display correctly on the page, avoiding sizing issues and forcing them to behave as block elements.
